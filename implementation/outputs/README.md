# Experiment Outputs

- `figures/`: selected plots, overlays, segmentation examples, and error-analysis panels suitable for the report.
- `metrics/`: compact CSV, JSON, or Markdown summaries of evaluation results.
- `predictions/`: bulk generated predictions; ignored by Git unless deliberately curated elsewhere.

Use stable, descriptive names that include the experiment or model identifier. Do not overwrite evidence from a reported run without recording the reason.
